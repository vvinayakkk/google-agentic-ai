PHASE 1 — SHARED LIBRARY & FOUNDATION
===================================================================

1.  shared/core/constants.py  [MODIFIED]
    • Added 14 new Firestore ref_* collections:
      ref_mandi_prices, ref_mandi_directory, ref_farmer_schemes,
      ref_soil_health, ref_equipment_providers, ref_cold_storage,
      ref_reservoir_data, ref_crop_varieties, ref_pmfby_data,
      ref_msp_prices, ref_fertilizer_data, ref_pesticide_advisory,
      ref_fasal_data, ref_pin_master
    • Added admin collections: admin_users, admin_audit_logs,
      app_config, analytics_snapshots
    • Added agentic collections: agent_conversations,
      notification_preferences, voice_sessions
    • Added ref_data_ingestion_meta for tracking data freshness
    • Added 6 Qdrant collection names:
      schemes_semantic, schemes_faq, mandi_price_intelligence,
      crop_advisory_kb, geo_location_index, equipment_semantic
    • Added SUPER_ADMIN role to UserRole enum
    • Updated EMBEDDING_DIM from 384 to 768

2.  shared/schemas/  [6 NEW FILES]
    • scheme.py   — SchemeSearchRequest, SchemeResponse,
                    EligibilityCheck, PMFBYData
    • market.py   — MarketPrice, PriceTrend, MSPPrice,
                    ColdStorage, Reservoir, MandiDirectory
    • agent.py    — ChatRequest, ChatResponse, ConversationMessage,
                    SessionSummary
    • admin.py    — AdminLoginRequest, AdminUserCreate,
                    AppConfigUpdate, AuditLog, DataFreshness,
                    AnalyticsOverview, FarmerStatusUpdate
    • geo.py      — PinLookupResponse, VillageSearchRequest,
                    DistrictList, StateList
    • notification.py — PriceAlert, NotificationPreferencesUpdate,
                    BroadcastRequest

3.  shared/schemas/__init__.py  [MODIFIED]
    • Exports all 6 new schema modules

4.  shared/core/config.py  [MODIFIED]
    • Added 4 new service URLs:
      SCHEMES_SERVICE_URL  (port 8009)
      GEO_SERVICE_URL      (port 8010)
      ADMIN_SERVICE_URL    (port 8011)
      ANALYTICS_SERVICE_URL (port 8012)
    • Removed deprecated CALENDAR_SERVICE_URL

5.  shared/auth/deps.py  [MODIFIED]
    • Admin users now looked up from admin_users collection
    • Added get_current_super_admin dependency
    • Updated get_current_admin for both ADMIN + SUPER_ADMIN

6.  shared/errors/codes.py  [MODIFIED]
    • Added error codes: SCHEME_NOT_FOUND, MARKET_DATA_NOT_FOUND,
      GEO_PINCODE_NOT_FOUND, ADMIN_USER_EXISTS,
      ANALYTICS_NOT_AVAILABLE, NOTIFICATION_LIMIT_EXCEEDED,
      QDRANT_ERROR

7.  shared/services/qdrant_service.py  [NEW]
    • Shared Qdrant vector service
    • Uses paraphrase-multilingual-mpnet-base-v2 (768-dim)
    • Lazy initialization of model + client
    • Methods: embed_text, embed_batch, search, ensure_collection,
      recreate_collection, upsert

PHASE 2 — NEW SCRIPTS
===================================================================

8.  scripts/seed_reference_data.py  [NEW]
    • Seeds ALL 14 ref_* Firestore collections from CSV files
    • Reads from scripts/reports/ subdirectories
    • Also seeds app_config with default global settings
    • Logs ingestion metadata to ref_data_ingestion_meta

9.  scripts/seed_admin.py  [NEW]
    • Creates 2 admin users (superadmin + admin)
    • Hashes passwords with bcrypt
    • Outputs credentials to audit CSV

10. scripts/generate_qdrant_indexes.py  [NEW]
    • Builds all 6 Qdrant vector collections from Firestore data
    • Uses multilingual-mpnet embeddings
    • Logs ingestion metadata

11. scripts/generate_analytics_snapshots.py  [NEW]
    • Generates daily analytics snapshots
    • Counts: total farmers, new today, DAU, agent queries,
      voice sessions, top commodities, notifications
